# NixOS 26.05 – Lenovo Legion Go

Modular **NixOS 26.05 configuration** for an AMD-based handheld
(Lenovo Legion Go–class hardware).

This repository contains **no secrets** and is safe to publish.

## Key Features
- NixOS 26.05
- KDE Plasma 6 (Wayland)
- AMD GPU (amdgpu + RADV)
- Gaming stack: GameMode, Gamescope, MangoHud
- Handheld Daemon (hhd)
- Home Manager via NixOS module

## Directory Structure (`/etc/nixos`)

- `flake.nix` — main flake file defining the system and its modules
- `configuration.nix` — main system configuration file
- `hardware-configuration.nix` — hardware-specific configuration generated by `nixos-generate-config`
- `boot.nix` — boot system configuration
- `desktop.nix` — graphical environment configuration
- `gaming.nix` — gaming-specific configuration
- `networking.nix` — network configuration
- `locale.nix` — locale and language settings
- `users.nix` — user configuration
- `packages.nix` — system package definitions

## Notes

- `hardware-configuration.nix` is hardware-specific and not versioned
- User passwords and SSH keys are configured locally
- Git uses SSH authentication
- We use Nix flakes for easy configuration management and rollback
- Each configuration area has its own file for better clarity and maintenance
- Packages and options are kept up-to-date with the latest available versions
- Comments in the files are written concisely and technically in Polish
